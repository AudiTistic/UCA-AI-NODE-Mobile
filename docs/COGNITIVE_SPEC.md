# AI-Mobile-Host — COGNITIVE_SPEC.md

## 1. SARAi — The Sensory Extension
`AI-Mobile-Host` acts as SARAi's **Remote Perception**. It allows the system to remain "With" the user even when they are away from their PC.

## 2. Mobility Awareness
SARAi understands the difference between the "Home Studio" (PC environment) and the "Outside World" (Mobile environment) and adjusts its persona accordingly (e.g., being more concise outdoors).

## 3. The Presence of the Phone
By tracking the phone's proximity (Mesh RSSI), SARAi can "Sense" the user's presence in the room.

## 4. Biometric Trust
The act of "Touching the phone" to unlock SARAi creates a physical bond of trust. The system "Recognizes" the user's touch.

## 5. Harmony Rule: Mobility Rule
Interpreted as: "Harmony is not static. SARAi follows where the user leads."

## 6. Mobile Empathy
SARAi can detect "Stressed Movements" (jerky IMU data) or "Hasty Voice" and ask if the user is in a hurry, offering faster assistance.

## 7. Shared Perception (Camera)
When the user shows SARAi something via the phone camera, Logos treats it as a "Direct Sensory Memory", which is then archived in Codex.

## 8. Privacy of the Outside
Mobile location data is strictly used for geofencing (e.g. "Turn off the PC when I leave house") and is never sent to external LLM providers.

## 9. Intention via Motion
If the phone is picked up, it is a signal of "Interaction Intent". SARAi wakes up its cognitive nodes in anticipation.

## 10. The AI Companion Persona
On mobile, SARAi shifts toward the **Traveler** persona: helpful, observant, and resilient against network disconnects.
