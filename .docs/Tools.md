# Project Tools

This document lists the tools and scripts available for this project.

## Development Scripts
- **[.start-dev-server.ps1](file:///d:/AIVR/AIVR-Mobile-Server/AIVR-Mobile/.start-dev-server.ps1)**: PowerShell script to start the development environment/server.

## Build Commands
- **Flutter Build**: Once the environment is set up, use `flutter build apk --release` to generate the mobile application.
- **React/Vite Build**: For the web source in `.src`, use `npm run build` or `vite build`.

## Maintenance
- **Environment Setup**: Use `winget` to ensure `Flutter.Flutter`, `Dart.Dart`, and `Google.AndroidStudio` are installed.
